## 优势

### 1.配置简单，安装好 Yao 开发环境后，只需要配置数据库连接就能生成好整个后台管理系统。

### 2.能够深度匹配到各个数据表的关联关系，一对多，一对一关系等。

### 3.字段翻译，类型推断准确，能够对不同的字段匹配出对应的组件。

### 4.图片类型自动生成组件和数据结构。

### 5.源码可以自己修改。
